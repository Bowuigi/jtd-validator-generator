# jtd-validator-generator

Generate Typescript data validators from [JSON Type Definition](https://jsontypedef.com/) / JTD / [RFC 8927](https://datatracker.ietf.org/doc/html/rfc8927) schemas.

The project is far from usable at the moment.

Its main purpose is serving as [Justify](https://github.com/Bowuigi/Justify)'s JSON format validator, replacing [Ajv](https://ajv.js.org/)'s JTD compiled output.
